My Web Development Documentation

A personal summary of what I've studied in HTML, CSS, and JavaScript.

HTML (HyperText Markup Language)

HTML (HyperText Markup Language) is the standard language used to create and design webpages. It structures content on the web, using elements like headings, paragraphs, links, and images. HTML uses tags to define and format content, forming the foundation of any website.

Key Concepts I Studied in HTML

  • Basic structure of an HTML document: <html>, <head>, <body>
  • Common tags: <p>, <h1>, <a>, <ul>, <img>
  • Forms: <form>, <input>, <textarea>, <button>
  • Attributes: id, class, href, src

Example: Basic HTML Page

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Welcome to My Page</h1>
    <p>This is a simple HTML page.</p>
  </body>
</html>

CSS (Cascading Style Sheets)

CSS (Cascading Style Sheets) is a style sheet language used to control the presentation of HTML elements. It defines the layout, colors, fonts, spacing, and overall design of web pages. CSS works through selectors to target HTML elements and apply properties like color, size, and position. Key concepts include the **box model**, which defines the content, padding, border, and margin of elements, and **positioning** methods like static, relative, absolute, and fixed. Additionally, **media queries** enable responsive design, adjusting the layout for different screen sizes. CSS enhances user experience by making websites visually appealing and adaptable.

Key Concepts I Studied in CSS

  • Selectors: Type selectors, class selectors, id selectors
  • Box model: margin, padding, border, and content
  • Colors: Using color names, hex codes, rgb, rgba
  • Positioning: static, relative, absolute, fixed, sticky
  • Responsive design: Using media queries to adjust styles for different screen sizes
  • Flexbox and Grid Layouts

Example: Styling a Webpage

body {
  background-color: #f4f4f4;
  font-family: Arial, sans-serif;
}

h1 {
  color: #3498db;
}

p {
  color: #333;
  font-size: 16px;
}

JavaScript

JavaScript allows me to make webpages interactive. I have learned how to manipulate elements, handle events, and use basic programming concepts.

Key Concepts I Studied in JavaScript

  • Variables: let, const, var
  • Functions: Creating reusable code blocks
  • Events: Handling events like clicks, form submissions
  • DOM Manipulation: Changing HTML elements dynamically
  • Conditionals and Loops: if-else, for, while loops
  • Arrays and Objects: Storing and accessing data

Example: JavaScript Function

function greet() {
  console.log('Hello, World!');
}

greet(); // Calling the function
