Day 5: Inheritance

  • Learned code reusability through inheritance.
  • Implemented a Person class and extended it with Student.
class Person { /* name, age */ }

class Student extends Person {
    String rollNo;
    Student(String name, int age, String rollNo) {
        super(name, age);
        this.rollNo = rollNo;
    }
}
